🎓 SchoolApp | Plateforme de Gestion Éducative
Une application web moderne et full-stack conçue pour faciliter la mise en relation entre enseignants et étudiants, la gestion des programmes scolaires et le suivi des performances académiques.

🚀 Fonctionnalités Clés
Système d'Authentification Complet : Connexion et Inscription avec gestion des rôles (Étudiant vs Invité).

Annuaire des Enseignants : Exploration des profils pédagogiques avec filtrage par matière.

Catalogue de Programmes : Visualisation détaillée des cours (prix en DH, nombre de leçons, durée).

Dashboard Étudiant : Accès sécurisé aux notes ("Mes Notes") pour les utilisateurs connectés.

Interface Responsive : Design fluide optimisé pour Mobile et Desktop avec Tailwind CSS.

Backend Robuste : API REST développée avec Django Rest Framework (DRF).

🛠️ Stack Technique
Frontend:

React.js

Tailwind CSS (Styling)

React Router DOM (Navigation)

Lucide React (Iconographie)

Backend:

Python / Django

Django REST Framework (API)

PostgreSQL (Base de données)
